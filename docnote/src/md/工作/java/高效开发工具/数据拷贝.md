## BeanUtils.copyProperties
一个浅拷贝方法，复制属性值
```
//原始复制
User user = new User();
user.setUsername(inputEntity.getUsername());
user.setAge(inputEntity.getAge());
...
﻿
//工具基本处理
User user = new User();
BeanUtils.copyProperties(inputEntity,user);

//工具基本处理（排除某些属性）
User user = new User();
BeanUtils.copyProperties(inputEntity,user,new String[]{"id","name"});
```
